228 Summary Ranges

评价

中等

题目

Given a sorted integer array without duplicates, return the summary of its ranges.

题目理解

找出数组中连续的数字段：因为数组是排序好的，所以我们可以从第一个数字开始，对数组中每个元素按顺序进行处理，设置两个数字用来表示目前的连续数字段，一个作为首，一个作为尾，当新遇到的数字与该数字段不连续时，将旧的数字段存储进结果中，将新遇到的数字作为新的数字段进行处理。

注意点

  1. 需要先判别nums数组是否为空，否则会出错

  2. 当迭代到最后一个元素，要思考最后一个元素有没有处理，有可能还没有存储进结果数组中

  3. 在c++中，数字变string可以用string库中的 to_string 函数
